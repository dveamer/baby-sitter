naver-cafe-tasks.md 파일에 기록된 커뮤니티 중에서 1개를 골라서 `두 번째 등급 기준` 을 만족시키기 위한 작업을 할거야. 

등업(등급 업그레이드)를 하기 위해서는 몇일간 출석이 필요하기도해서 한번에 작업을 완료할 수 없고 여러번의 스케쥴에 의한 실행을 통해서 업그레이드 달성을 하는 것이 목표야. 
한번 작업에 5개 커뮤니티에 대해서 처리할 수 있는 최대한의 작업을 진행해줘. 

출석 조건만 남은 커뮤니티의 경우에는 출석은 계속 해주되 5개 커뮤니티 개수에는 포함하지 말아줘. 
A, B 커뮤니티는 출석만 해도 되는 상황이면 A, B는 출석만 해주고 C, D, E, F, G 7개의 커뮤니티의 댓글, 글 등의 기준 달성을 위한 작업을 진행해줘.

# 로그인 팁

mcp openchrome 을 이용해서 브라우저로 네이버 로그인을 시도해줘. 실제 로그인시 필요한 아이디, 패스워드 입력은 내가 진행할게. 

# 가입하기 

가입이 안되어있는 곳은 사람이 직접 가입을 해줘야할거야. 
진행하다가 가입이 안되어있어서 진행이 불가하다면 다른 카페를 먼저 진행해주고 가입이 필요한 카페들은 별도로 분류해서 내게 알려줘. 

# 출석 팁

로그인이 완료된 상태면 "가입 후 등업작업 중" 인 대상의 카페들은 가장 먼저 무조건 1회씩 방문해줘.

# 현황 파악 팁

네이버에 로그인을 한 상태에서 "멤버등급 안내" 에 접속하면 "총 게시글 0개, 댓글 수 0개, 좋아요 0개, 방문 수 0회, 가입일 0' 과 같은 해당 카페에서의 내 활동 정보가 제공될거야. 

# 댓글 등록 팁 

가입인사 게시판이 있다면 다른 사람의 가입인사 글에 "안녕하세요~", "반갑습니다~" 와 같이 간단한 인사 댓글만 남겨도 쉽게 목표치를 달성할 수 있을거야. 

# 글 등록 팁

현재 등급에서 글을 남길 수 있는 게시판을 찾는 것이 가장 어려워. 

일단 가입인사 게시판에 간단하게 "안녕하세요 잘 부탁드립니다~!" 정도의 가입인사를 남기는 것만으로도 1건을 등록할 수 있어.
가입인사를 여러번 남기면 이상하니까 가입인사 게시판에는 1개의 글만 적어줘. 

육아/육아템/육아용품 관련 질문을 남길 수 있는 게시판이 있다면 아래 같은 질문을 남겨도 좋아.
  * 홈캠 어떤 것 쓰세요?
  * 12개월 이전에도 분리 수면 가능한가요?
  * 슬링 아기띠 어떤 것 사야할까요? 당근해도 괜찮을까요? 
  * 홈캠 반드시 필요할까요?

# 결과 처리 

등급이 업그레이드 되었다면 naver-cafe-result.md 파일에 카페 주소를 기록해줘. 
기록된 카페 주소는 재처리 되지 않도록 naver-cafe-tasks.md 에서 제거해줘. 

# 작업 종료 
 
작업이 모두 완료되었다면 사용했던 chrome-devtools(openchrome) 은 종료시켜줘.


## 주의 사항

chrome-devtools(openchrome) 사용에 문제가 있다면 아래 같은 문제 점이 예상돼.
  1. chrome-devtools(openchrome) 에서 네이버 로그인이 되어있지 않아서 
  2. 이전에 사용했던 chrome-devtools(openchrome) 리소스가 정리가 되지 않아서 
  
1번이 원인일 경우에는 대기해서 내가 로그인해주는 수밖에 없어. 
2번이 원인일 경우에는 chrome-devtools(openchrome) 리소스를 종료 시키면 다음 스캐쥴에서는 정상적으로 동작할 것으로 예상돼

문제점이 생겼을 때 Playwright 를 사용해서 해결해보려고하지마. 성공하는 것이 기록된 적이 없어.
