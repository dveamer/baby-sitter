
# https://babysitter.dveamer.com 소개 댓글 준비

분리수면 관련 주제를 가진 네이버블로그(blog.naver.com) 글들을 방문해서 댓글에 어울릴만한 소개문구를 준비할거야. 
  * 주제 : 분리수면, 수면교육, 수면독립

본문의 내용 중 분리수면을 잘하기 위해서는 부모가 참아야한다는 내용의 문구를 한두개 정도 인용해서 공감을 표현을 하고 babysitter 앱에 대한 소개를 적어줘. 

## 문구 예시

아래는 작성할 문구 예시야. 

```
예시 1 사이트 : https://blog.naver.com/aroundsu/224267789260

안녕하세요. 아기 수면 보조 앱 babysitter를 만들고 있는 팀입니다.
글에서 말씀하신 "아기가 울면 바로 들어가지 말고 기다리기", "부모 개입 줄이기" 라는 부분에 많이 공감했습니다.

저희는 방문을 바로 열지 않아도 부모 목소리 자장가를 들려주면서 조금 더 기다려볼 수 있게 돕는 기능을 만들고 있습니다.
자세한 내용은 https://babysitter.dveamer.com 에서 보실 수 있습니다. 
이 앱이 분리수면을 시작하는 부모들에게 도움이 될까요?
```


## 결과 기록

promotion-tasks.md 파일에 위에 내가 공유해준 예시처럼 글을 남긴 사이트 주소와 남긴 댓글 내용을 추가해줘. 각각을 하나의 task 처럼 구분해서 작성해줘. 

## 기처리건 대상 제외

promotion-results.md 파일은 이미 처리가 완료된 건들이 기록되어있어. 해당 글들에 대해서는 댓글 준비를 하지 말아줘.
